### Usage :

-> run 

_For run the Project use the cmdlet command into the folder_

> npx serve . or > npx --yes serve .

*If don't wanna use a command, use a **live viewer extention***


for admin usage
```
> user: superadmin@nexoavionix.com
> pass: any
```
for agent usage
```
> user: agente@nexoavionix.com
> pass: any
```

for user usage
```
> user: any@mail.domain
> pass: any
```
