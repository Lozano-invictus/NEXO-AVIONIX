# Imágenes de Destinos - Nexo Avionix

Coloca aquí las imágenes de los destinos turísticos. Formatos recomendados: JPG, WebP (optimizadas para web).

## Convención de nombres

- `cartagena.jpg` - Cartagena (CTG)
- `san-andres.jpg` - San Andrés (ADZ)
- `medellin.jpg` - Medellín (MDE)
- `bogota.jpg` - Bogotá (BOG)
- `cali.jpg` - Cali (CLO)
- `santa-marta.jpg` - Santa Marta (SMR)

## Tamaños recomendados

- Tarjetas de destino: 800x600px (relación 4:3)
- Hero banner: 1920x1080px o 1920x800px
- Optimizar para web (compresión 80-85%)

## Fuentes de imágenes gratuitas

- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)

---

**Nota**: Si no hay imágenes disponibles, el sistema mostrará gradientes de color como respaldo.
